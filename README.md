# Simple Interest Calculator

A shell script to calculate simple interest based on principal, annual rate of interest, and time period in years.

## Description
This project provides a simple utility to compute interest amounts. It is designed to be used in a terminal environment.

## Formula Used
The calculation follows the standard formula:
`Simple Interest = (P * R * T) / 100`
Where:
* **P** = Principal amount
* **R** = Annual rate of interest
* **T** = Time period in years

## Usage
1. Make the script executable: `chmod +x simple-interest.sh`
2. Run the script: `./simple-interest.sh`

## Example
**Input:**
* Principal: 1000
* Rate: 5
* Time: 2

**Output:**
* Simple Interest: 100
